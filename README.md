# Customer Insights Dashboard

## Overview
Analysed customer complaint data to identify trends, improve service performance, and support data-driven decision-making.

## Dashboard Preview
![Dashboard](dashboard.png)

## Tools Used
- Excel
- Data Analysis
- Data Visualisation

## Key Insights
- Payment Issues and Delivery Delays are the most frequent complaint types
- Average resolution time is approximately 5 days
- Majority of complaints are resolved, but open issues indicate process gaps

## Business Impact
- Helps prioritise high-impact issues
- Supports faster resolution strategies
- Improves customer satisfaction and operational efficiency

## Skills Demonstrated
Data Analysis | KPI Reporting | Problem Solving | Business Insights
